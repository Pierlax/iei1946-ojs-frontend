# Deployment Guide - Economia Internazionale Frontend\n\n## Overview\n\nQuesto frontend è costruito con **React 19 + Vite** e ottimizzato per il deployment su **Vercel**. Il sito si interfaccia con un backend **OJS 3.4** tramite REST API.\n\n## Prerequisites\n\n- Node.js 18+ e pnpm\n- Account Vercel (https://vercel.com)\n- GitHub repository (per il deployment automatico)\n- OJS 3.4 installato e configurato con API REST abilitata\n\n## Local Development\n\n```bash\n# Installazione dipendenze\npnpm install\n\n# Avvio server di sviluppo\npnpm dev\n\n# Build per produzione\npnpm build\n\n# Preview della build\npnpm preview\n```\n\n## Deployment su Vercel\n\n### Opzione 1: Deployment Automatico (Consigliato)\n\n1. **Push del codice su GitHub**\n   ```bash\n   git init\n   git add .\n   git commit -m \"Initial commit: Economia Internazionale frontend\"\n   git branch -M main\n   git remote add origin https://github.com/yourusername/iei1946-nextjs-frontend.git\n   git push -u origin main\n   ```\n\n2. **Collegamento a Vercel**\n   - Visita https://vercel.com/new\n   - Seleziona il repository GitHub\n   - Vercel rileverà automaticamente la configurazione Vite\n   - Clicca \"Deploy\"\n\n3. **Configurazione Variabili d'Ambiente**\n   - Nel dashboard Vercel, vai a Settings > Environment Variables\n   - Aggiungi le seguenti variabili:\n     - `REACT_APP_OJS_BASE_URL`: URL del tuo backend OJS (es. https://ojs.iei1946.it)\n     - `REACT_APP_OJS_JOURNAL_PATH`: Percorso della rivista in OJS (es. iei)\n\n### Opzione 2: Deployment Manuale\n\n1. **Installazione Vercel CLI**\n   ```bash\n   npm i -g vercel\n   ```\n\n2. **Login e Deploy**\n   ```bash\n   vercel\n   ```\n\n3. **Seguire le istruzioni interattive**\n\n## Configurazione OJS API\n\nAssicurati che il tuo backend OJS abbia:\n\n1. **REST API abilitata**: In OJS, vai a Settings > Website > Plugins e abilita \"REST API Plugin\"\n2. **CORS configurato**: Se il frontend è su un dominio diverso da OJS, configura CORS nel file `config.inc.php` di OJS:\n   ```php\n   ; In config.inc.php\n   allow_cors = On\n   cors_allowed_hosts = \"https://tuo-dominio-vercel.vercel.app\"\n   ```\n\n## Endpoint OJS API Utilizzati\n\nIl frontend utilizza i seguenti endpoint OJS:\n\n- `GET /api/v1/journals/{journalPath}/articles` - Lista articoli\n- `GET /api/v1/journals/{journalPath}/articles/{articleId}` - Dettagli articolo\n\n## Performance Optimization\n\n### Vercel Edge Functions (Opzionale)\n\nPer migliorare ulteriormente le prestazioni, puoi aggiungere Edge Functions per il caching:\n\n```typescript\n// api/cache-articles.ts\nexport const config = {\n  runtime: 'edge',\n};\n\nexport default async function handler(request: Request) {\n  const url = new URL(request.url);\n  const ojsUrl = process.env.REACT_APP_OJS_BASE_URL;\n  \n  const response = await fetch(`${ojsUrl}/api/v1/journals/iei/articles`);\n  const data = await response.json();\n  \n  return new Response(JSON.stringify(data), {\n    headers: {\n      'Content-Type': 'application/json',\n      'Cache-Control': 'public, s-maxage=3600, stale-while-revalidate=86400',\n    },\n  });\n}\n```\n\n## Monitoring\n\nVercel fornisce automaticamente:\n\n- **Web Analytics**: Visualizza metriche di performance (Core Web Vitals)\n- **Error Tracking**: Traccia errori del frontend\n- **Deployment Logs**: Monitora i log di build e runtime\n\nAccedi al dashboard Vercel per visualizzare queste metriche.\n\n## Troubleshooting\n\n### Errore: \"OJS API not reachable\"\n\n1. Verifica che l'URL OJS sia corretto nelle variabili d'ambiente\n2. Controlla che CORS sia configurato correttamente in OJS\n3. Verifica che l'API REST sia abilitata in OJS\n\n### Build Failure\n\n1. Verifica i log di build nel dashboard Vercel\n2. Assicurati che tutte le dipendenze siano corrette in `package.json`\n3. Esegui `pnpm install` e `pnpm build` localmente per testare\n\n### Performance Issues\n\n1. Usa Vercel Analytics per identificare i colli di bottiglia\n2. Implementa caching per le richieste API OJS\n3. Considera l'uso di Edge Functions per il caching\n\n## Dominio Personalizzato\n\nPer collegare un dominio personalizzato (es. iei1946.it):\n\n1. Nel dashboard Vercel, vai a Settings > Domains\n2. Aggiungi il tuo dominio\n3. Segui le istruzioni per configurare i DNS record\n\n## SSL/TLS\n\nVercel fornisce automaticamente certificati SSL/TLS gratuiti per tutti i domini.\n\n## Support\n\nPer domande o problemi:\n\n- Vercel Docs: https://vercel.com/docs\n- OJS API Documentation: https://docs.pkp.sfu.ca/dev/api\n- GitHub Issues: Apri un issue nel repository\n
